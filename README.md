# F1-visualizer

F1-Visualizer was submitted to .devHacks 2024, hosted by .devClub at the University of Manitoba. The challenge was to develop an application within a 24-hour period. F1-Visualizer was awarded the Open Innovation Challenge Hack.

This application calls an API to retrieve racing data and visualizes it for the user using the MonoGame engine.
